# 🏦 Banking Risk Analytics & Dashboarding

## 📌 Project Overview

This project is an end-to-end Data Analytics solution focused on analyzing banking customer data to assess credit risk and minimize financial lending losses. It combines Python-based data analysis, SQL database management, and Power BI visualization to generate actionable business insights.

---

## 🎯 Objective

To identify high-risk and low-risk customer segments using data-driven techniques and build interactive dashboards that support financial decision-making in banking and risk management.

---

## 🛠️ Tech Stack

* **Python**: Pandas, NumPy, Matplotlib, Seaborn
* **SQL**: MySQL (Data Storage, Querying, Data Management)
* **Power BI**: Dashboarding, Data Modeling, DAX
* **Statistics**: Risk Analysis, Customer Segmentation

---

## 🔄 Project Workflow

### 1. Data Processing & Cleaning

* Handled missing values, duplicates, and inconsistent data
* Standardized and transformed raw datasets for analysis

### 2. Exploratory Data Analysis (EDA)

* Analyzed customer demographics, income groups, loan status, and repayment behavior
* Generated visual insights using Matplotlib and Seaborn
* Identified patterns in high-risk vs low-risk customers

### 3. Statistical Analysis

* Applied statistical methods to evaluate credit risk factors
* Segmented customers based on financial behavior and risk profile

### 4. Database Management (MySQL)

* Designed relational database structure for banking data
* Performed SQL operations for efficient data retrieval and reporting

### 5. Data Visualization (Power BI)

* Built interactive multi-page dashboards
* Visualized KPIs such as loan exposure, deposits, and customer segmentation
* Used DAX for calculated measures and dynamic reporting

---

## 📊 Key Insights

* Identified customer groups with higher credit risk
* Improved understanding of loan repayment behavior
* Enabled data-driven financial decision-making through dashboards
